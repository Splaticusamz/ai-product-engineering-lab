# AI Product Engineering Lab

Practical experiments for building AI-assisted products that feel useful, not gimmicky.

This repo is a public working notebook for small, shippable artifacts around:

- agentic product workflows
- UX patterns for AI features
- evaluation and guardrail notes
- Next.js / React / TypeScript implementation patterns
- local automation scripts that turn vague ideas into repeatable systems

The goal is not streak farming. The goal is a visible trail of useful product-engineering judgment: small experiments, clear writeups, and code that another builder can reuse.

## Structure

```txt
experiments/       One experiment per topic with notes, scripts, and results
patterns/          Reusable product/UX/engineering patterns
scripts/           Utility scripts used by experiments
logs/              Dated progress logs with what changed and why
```

## Current focus

1. AI-native product surfaces that preserve user control.
2. Lightweight evals for agent workflows.
3. High-signal automation that creates real artifacts instead of noise.

## Operating rule

Every automated contribution should contain at least one of:

- a runnable script
- a concrete product pattern
- a measured result
- a useful critique/checklist
- a reproducible experiment note

No empty commits. No timestamp churn. No fake activity.
